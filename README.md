# RepoPortifolio

#  Portfólio Profissional
 Este é um portfólio moderno desenvolvido para apresentar meus projetos como desenvolvedor web, um projeto fictício desenvolvido com o objetivo de praticar criação de interfaces web, organização de arquivos e boas práticas de versionamento utilizando Git e GitHub. O site simula um portfólio profissional de arquitetura, apresentando projetos, imagens ilustrativas e uma estrutura simples em múltiplas páginas.

O foco deste projeto é exclusivamente educacional, servindo como exercício para HTML, CSS, design responsivo e publicação através do GitHub Pages.

##  Tecnologias Utilizadas
   HTML5 - CSS3 - Git e GitHub - GitHub Pages 

##  Páginas do Projeto
`index.html` — Página inicial com apresentação profissional - `projeto.html` — Cards animados com meus projetos
